Welcome to intergalactics!

To play, you'll need to do one of these things:

  • Go to http://intergalactics.net, and play over the web. This is the easiest way to get started, so I recommend this first. Requires Internet Explorer 4.0 (or greater), or Netscape 4.5 (or greater).
  • If you have Windows 98 or Internet Explorer 4 (or greater) then you're all set. Make sure you connected to the Internet, and type:

       intergalactics ie intergalactics.net

    (Or substitute a different server for intergalactics.net)

  • Otherwise, you'll have to download the Java Runtime Environment from Sun. Go here to do that: http://www.java.sun.com/products/jdk/1.2/jre/index.html. Then type:

       intergalactics intergalactics.net

Read the online docs available at http://www.cs.utoronto.ca/~jw/intergalactics for further assistance.

Here's a brief explanation of what you're looking at:

  • This graph is the result of an enormous round-robin tournament where for each galaxy, a game was played using every possible permutation of robots over the home planets. This is the most thorough method of testing robot-to-robot combat I could think of!
  • The Rank column indicates the overall ranking of each robot. Win% is the percentage of games the robot won.
  • Each column under the Game Size % category represents the win percentage of the robot in games with the given number of players.
  • Lastly, the columns corresponding to other robots indicate the relative number of wins versus losses scored with that robot.
